```cpp
Функции передают копию, лучше использовать указатель

int a = 1;  
int& a_ref = a;
conts int& x = a;  (через неё нельзя изменить a)

Передавая в функцию неконстантную ссылку можно изменить объект.

int& = 1 + 2;  (так нельзя, 3 умирает)
const int& = 1 + 2;  (так можно)


К временным объектам можно привязать только константные ссылки,
ссылка указывает на область памяти и не знает жива ли переменная.
Константная ссылка продлевает жизнь временного объекта внутри 
области своей видимости.

Указатель:
Указатель хранит индекс ячейки памяти

int n = 1;
int* n_pointer = &n;  (дает адрес ячейки с n)
				*n_pointer = 1  (обращение к ячейке памяти по индексу)

const int* n; 
int* const n;  (константный указатель)

int* x = new int(5);
delete x;

pointer + 1    (указывает на следующий объект в памяти)
```
